# TeCoD Documentation

This documentation covers the public TeCoD runtime package.

- [Quickstart](quickstart.md): install, prepare data, and run TeCoD.
- [Configuration](configuration.md): Hydra profiles, overrides, and environment variables.
- [Data preparation](data-preparation.md): expected files and preparation commands.
- [API reference](api.md): Python API, request models, and output fields.
- [Generation methods](generation-methods.md): `auto`, `gcd`, `base-gcd`, `sgc`, `icl`, and `zs`.
- [Architecture](architecture.md): services, data flow, and runtime components.

Clone with submodules before following the guides:

```bash
git clone --recurse-submodules https://github.com/SSLab-CSE-IITB/tecod.git
```

The project page for the paper is in [`../site/index.html`](../site/index.html).
